Dict

The Dict class implements a dictionary. The keys should implement a 'hash()' method. If the key is not hashable an id generated by 'uniqid()' will be used instead. The keys may be mutable but must manually be updated when modified (remove, modify, add). Otherwise the Dict will break.
Genewrapor

The class is intended for wrapping generators but it can handle all types of iterables (implementing the Iterator interface). That way generator functionaliy can be chained from different code parts (just like functions). Also optionally this class makes it possible to reiterate over a generator.
